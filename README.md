# Inception

**System Administration Project**

An automated environment setup using Docker and Docker Compose. This project orchestrates multiple services to simulate a realistic web infrastructure including:
- nginx
- mariaDB
- wordpress
The project is to be cloned on a virtual machine. Makefile copies the secrets and .env file from host computer via shared folder. 
There is also .env template included within the project.
Use make help to show options.
Each continer has its own README for more info and explaining the key concepts.

Score 100/100
